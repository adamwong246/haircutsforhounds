# St Vincent's Haircut's For Hounds

This project is a webpage for a non-profit organization Haircuts For Hounds.

## Theme

stained glass window of Catholic Saint wearing baseball cap, with short black hair and short black beard. he is prominently holding a pair scissors. he is surrounded by happy dogs

## Color Theme: "Stained Glass Saint"

The color palette is derived from the stained glass window theme described above.

### Color Associations

| Element | Color | CSS Variable | Rationale |
|---------|-------|-------------|-----------|
| **Stained glass background** | Deep Cathedral Blue `#1a237e` | `--color-primary-dark` | Traditional church stained glass uses deep blues as the foundational color |
| **Gold leaf / halos** | Warm Gold `#d4a017` | `--color-gold` | Gold leaf is common in religious iconography and stained glass |
| **Baseball cap** | Ruby Red `#8b0000` | `--color-ruby` | A bold red accent for the modern baseball cap element |
| **Saintly reverence** | Amethyst Purple `#6a0dad` | `--color-amethyst` | Purple symbolizes royalty and spirituality in religious contexts |
| **Hope / growth (non-profit mission)** | Emerald Green `#2e7d32` | `--color-emerald` | Green represents growth, hope, and the non-profit's mission |
| **Dog fur** | Warm Brown `#8B4513` | `--color-warm` | Earthy brown for the happy dogs surrounding the saint |
| **Scissors** | Silver Metallic `#c0c0c0` | `--color-silver` | Metallic silver for the prominently held scissors |
| **Parchment / aged glass** | Warm White `#f5f5f0` | `--color-bg-light` | Light background mimics aged parchment or frosted stained glass |

### Semantic Usage

- **Links:** Gold (unvisited) → Amethyst (visited) → Ruby (hover)
- **Success/Donate:** Emerald Green
- **Warning/Events:** Ruby Red
- **Code blocks:** Dark green background with green text (terminal aesthetic)
- **Navigation & Footer:** Deep Cathedral Blue with warm white text

### Dark Mode

In dark mode, the background shifts to a deep navy `#1a1a2e` with gold text for headings, maintaining the stained glass atmosphere while ensuring readability.
